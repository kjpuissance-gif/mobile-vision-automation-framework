
# 🤖 Mobile Vision Automation Framework (MVAF)

**AI-Powered Visual Testing & Automation for Android Applications**

> Automate visual testing, UI validation, and repetitive tasks on Android devices using Computer Vision, LLM decision-making, and ADB control.

---

## 🎯 What It Does

MVAF turns your Android device into a fully scriptable environment. It "sees" the screen, understands the interface, and executes precise touch interactions — all without needing source code access or special permissions.

**Perfect for:**
- 🎮 **Game Studios** — Automated QA testing, UI regression checks, compatibility testing across devices.
- 🛒 **E-commerce Apps** — Cart flow validation, payment screen testing, onboarding verification.
- 📱 **App Agencies** — Client deliverable validation, demo automation, screenshot generation.

---

## ⚡ Key Features

- **🔍 Multi-Engine Vision :** Grounding DINO + YOLOv8 + OCR + HSV heuristics with Kalman filtering.
- **🧠 LLM Decision Layer :** Groq-powered AI decides next actions based on visual state.
- **📱 Universal ADB Control :** Works on any Android device, any resolution, no root needed.
- **🔄 Intelligent Fallbacks :** If one vision model fails, others take over seamlessly.
- **💬 Interactive Mode :** Chat with the AI to guide testing scenarios in real-time.

## 🚀 Quick Start

### Prerequisites
```bash
# Ubuntu/Debian
sudo apt-get install android-tools-adb
```

### Installation
```bash
git clone [https://github.com/kjpuissance-gif/mobile-vision-automation-framework.git](https://github.com/kjpuissance-gif/mobile-vision-automation-framework.git)
cd mobile-vision-automation-framework
pip install -r requirements.txt
```

### Configuration
```bash
echo "GROQ_API_KEY=your-api-key-here" > .env
python the_gamer_ia_god.py
```
